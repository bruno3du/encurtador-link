# Encurtador de Link

Este é um projeto back-end básico, onde trabalha com o rest api para criar um encurtador de links. 

Neste projeto foi utilizado MongoDB como banco de dados da aplicação.

Utilizei verificação para saber se link enviado segue modelo URL e evitando erro de redirecionamento.

## 🚀 Começando

Essas instruções permitirão que você obtenha uma cópia do projeto em operação na sua máquina local para fins de desenvolvimento e teste.

Consulte **Instalação** para saber como implantar o projeto.

### 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/). 
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)


### 🔧 Instalação

```bash
# Clone este repositório
$ git clone <https://github.com/bruno3du/encurtador-link.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd encurtador-link

# Instale as dependências
$ yarn install

# Verifique se colocou os dados do seu banco dados antes.

# Execute a aplicação em modo de desenvolvimento
$ yarn dev

# O servidor inciará na porta:3000 - acesse <http://localhost:3000>
```
